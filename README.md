# zshrcTmuxVimrc

文本三剑客的配置文件

* zsh
* tmux
* vim
* bullet-train(oh-my-zsh主题的自定义)
	[github.com/bullet-train](https://github.com/caiogondim/bullet-train.zsh)

分别将三个配置文件重命名放到家目录
`tmux.conf` -> `.tmux.conf`
`vimrc` -> `.vimrc`
`zshrc` -> `.zshrc`
`bullet-train.zsh-theme` -> ~/.oh-my-zsh/themes/bullet-train.zsh-theme
